# Templates-Nature-Figures

SVG figures templates for Nature like journals.

Two columns 2 x 2 and 1 x 3 are 183mm x 247mm.

Single-column 3 x 1 is 89mm x 247mm.

These templates are starting points to be futher defined according to the journal requirements. 

The measure of the page were found at https://research-figure-guide.nature.com
